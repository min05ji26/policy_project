# AI 에이전트 활용 청년 정책 도우미

> 단순 정책 나열이 아닌, 실제로 유리한 정책을 판단하는 서비스

AI 챗봇과 자연어로 대화하면 내 조건(나이·소득·거주형태 등)을 분석해,  
**실질 수혜 가능성이 높은 순서**로 청년 정책을 추천해드립니다.

---

## 배경 및 목적

청년을 대상으로 하는 정책은 많지만, 실제 수혜 구조는 청년에게 불리하게 설계된 경우가 많습니다.  
예를 들어 주택 청약 가점제는 무주택 기간·부양가족 수·납입 기간으로 점수를 산정하는데, 20대는 이 세 항목 모두 구조적으로 낮을 수밖에 없습니다.

이 프로젝트는 이런 **구조적 역설**을 데이터로 포착하고, 사용자 개인 조건을 AI가 분석해 실질 수혜 가능성을 예측합니다.

## ML 모델

### 설계 방식

KOWEPS(한국복지패널) 14,915명 데이터와 주거실태조사를 결합해 학습했습니다.

**Wide → Long format 전환**  
정책마다 별도 모델을 만들지 않고, `(사람 ID + 정책 ID)` 쌍으로 행을 변환해 단일 모델로 모든 정책을 통합 학습합니다. 정책 구분은 OHE(원-핫 인코딩) 컬럼 `pid_policy_{정책ID}`로 처리합니다.

**클래스 불균형 처리**  
수혜 21.4% vs 미수혜 78.6%의 불균형 → SMOTE 오버샘플링 적용

**피처 구성**
- 수치형: 소득, 나이, 가구원 수, 재산, 임차료 등
- 범주형: 고용형태, 거주형태, 혼인 여부 등 (LabelEncoder)
- 정책 OHE: `pid_policy_{정책ID}` 원-핫 인코딩

### 정책별 모델 성능

| 정책 | ROC-AUC | Accuracy | Precision | Recall | F1 |
|------|---------|----------|-----------|--------|----|
| 공적연금 | **0.80** | 0.689 | 0.430 | 0.916 | 0.585 |
| 생계급여 | 0.75 | 0.824 | 0.212 | 0.517 | 0.301 |
| 주거급여 | 0.74 | 0.793 | 0.238 | 0.559 | 0.334 |
| 의료급여 | 0.73 | 0.791 | 0.185 | 0.564 | 0.278 |
| 주거급여임차 | — | — | — | — | 수혜자 12명으로 학습 불가 |

> Precision이 낮아 비수혜자를 수혜자로 예측하는 오류가 일부 존재합니다.  
> Recall을 높이는 방향으로 설계해, 수혜 가능성 있는 사람을 놓치지 않는 것을 우선합니다.

## 사용자 흐름

```
[홈 화면]
  정책 카드 탐색 (탭: 전체 / 청년 / 주거 / 취업지원 / 저소득)
  ↓ "AI 챗봇으로 시작하기" 클릭
  ※ 로그인 없이도 분석 이용 가능

[채팅 화면]
  AI가 아래 9가지 항목을 자연어로 순차 질문
  나이 · 가구원 수 · 혼인 여부 · 거주형태 · 무주택 여부 ·
  월 소득 · 재산 · 고용 여부 · 부채 여부
  → 우측 사이드바에 수집 현황 실시간 표시 (예: 8/9)
  ↓ 9가지 정보 수집 완료 시 자동으로 분석 시작

[분석 처리]
  1. Gemini: 대화에서 정형 데이터 추출
  2. 규칙 엔진: policies_meta.json 자격요건 검사
  3. ML 모델: 정책별 수혜 확률 계산
  4. 계산기: 법령 기반 예상 수혜금액 산출

[결과 화면]
  전체 수혜 가능성 % 표시
  정책별 확률 바 차트 (높은 순 정렬)
  ↗ 정책별 신청 사이트 바로가기
  ⚠ 자격은 있으나 수혜율 낮은 역설 케이스 별도 표시
  ↓ 로그인 시 결과 저장 가능

[로그인 / 마이페이지] (선택)
  이메일 또는 카카오 소셜 로그인
  프로필(이름·출생연도·성별) 확인 및 수정
```

## API 엔드포인트

### 채팅 / 예측
| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/api/chat` | 멀티턴 대화, 사용자 정보 수집 |
| POST | `/api/predict` | 수집된 정보로 전체 정책 추천 |
| POST | `/recommend` | 자연어/프로필로 직접 추천 |
| DELETE | `/chat/{session_id}` | 세션 초기화 |

### 정책 조회
| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/policies` | 전체 정책 목록 |
| GET | `/policies/{id}/detail` | 정책 상세 + 샘플 수혜금액 |
| POST | `/policies/{id}/detail` | 내 정보 기반 맞춤 분석 |

### 인증
| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | `/auth/signup` | 이메일 회원가입 |
| POST | `/auth/login` | 이메일 로그인 |
| POST | `/auth/kakao` | 카카오 로그인 |
| GET | `/auth/me` | 내 정보 조회 |
| PUT | `/auth/me` | 내 정보 수정 |

## 시작하기

### 백엔드

```bash
cd back
pip install -r requirements.txt

cp .env.example .env
# .env에 GEMINI_API_KEY, JWT_SECRET 입력

uvicorn main:app --reload
# → http://localhost:8000
```

### 프론트엔드

```bash
cd front
npm install
npm run dev
# → http://localhost:5173
```

### 환경변수 (.env)

```env
GEMINI_API_KEY=your_gemini_api_key
JWT_SECRET=your_secret_key
```

## 한계 및 향후 계획

**현재 한계**
- 개인정보 보호 이슈와 공개 데이터의 한계로 **주거·복지 분야** 데이터에 한정

- Precision이 낮아 비수혜자를 수혜자로 예측하는 오류 일부 존재

**향후 확장 계획**
- 데이터 확보 시 청년 취업·교육·금융 정책으로 범위 확장
- 모델 고도화 (앙상블, XGBoost 등)
- 신청 결과 저장 및 이력 관리 기능
